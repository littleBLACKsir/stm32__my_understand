# 关于stm32的学习路线及规划

（取自牙大金学长）

## 1.第一项：

- 熟悉新建工程：

- 熟悉掌握stm32新建标准工程模板
- 理解stm32的多文件编程思想
- 熟悉MDK5的各种配置
- 学会多种烧录代码的使用
- J-link ，ST-link ，ISP串口等

## 2.第二项：

- 掌握GPIO控制

  * 八种模式
      > + 上拉输入
      > + 下拉输入
      > + 开漏输出
      > + 推挽输出
      > + 推挽复用功能
      > + 开漏复用
      > + 浮空输入  

- 掌握技能

  * 输出实验

    > **led点灯**
    > + 流水灯
    > + 继电器控制
    > + 频闪灯
    > + 2进制、16进制逻辑点灯

 * 蜂鸣器实验

 * 输入按键实验
 
      > + 按键输入
      > + 独立按键输入
    
  

## 3.第三项：

- 理解stm32串口工作原理，理解收发过程
- 理解各个寄存器的功能及其配置，掌握使用标准库，熟悉掌握函数的参数配置。
- 理解例程的文件逻辑框架，掌握多文件编程。3.
  -  熟练掌握配置一般流程掌握技能
   > + 板载usart2串口实验
   > + hc06蓝牙模块串口实验
   > + usart1、usart3实验

## 4.第四项：

- 掌握外部中断

- 重点理解中断的概念，理解中断事件的工作原理	

- 掌握中断优先级概念，理解各个寄存器的功能及其配置，掌握使用标准库，熟悉掌握函数的参数配置。

- 了解GPIO和中断线的映射图

- 掌握外部中断设置的一般流程
  （以精英板为例）

  > + 初始化 IO 口为输入
  > + 开启 AFIO 时钟
  > +  设置 IO 口与中断线的映射关系。
  > + 初始化线上中断，设置触发条件等。
  > + 配置中断分组（NVIC），并使能中断。
  > + 编写中断服务函数。

## 5.第五项

- 理解stm32定时器原理，掌握计算分频系数

- 了解通用定时器，了解各个定时器的一般用途

- 了解三种计数方式

- 向上计数

- 向下计数

- 向上/向下计数

- 理解各个寄存器的功能及配置，掌握使用标准库，熟悉掌握函数的参数配置。

   - 掌握定时器设置的一般流程
     （以精英板为例）

     	 > + TIM3 时钟使能。
     	 > +  初始化定时器参数,设置自动重装值， 分频系数，计数方式等。
     	 > + 设置 TIM3_DIER 允许更新中断。
     	 > +  TIM3 中断优先级设置。
     	 > +  允许 TIM3 工作，也就是使能 TIM3。
     	 > +  编写中断服务函数。

##  6.第六项：

- 理解PWM原理，重点理解占空比

- 了解stm32下pwm模式 ，了解相关参数，了解引脚重映射

- 理解各个寄存器的功能及其配置，掌握使用标准库，熟悉掌握函数的参数配置。

  - 掌握定时器PWM设置的一般流程
    （以精英板为例）

    	> + 开启 TIM3 时钟以及复用功能时钟，配置 PB5 为复用输出。
    	> +  设置 TIM3_CH2 重映射到 PB5 上。
    	> +  初始化 TIM3,设置 TIM3 的 ARR 和 PSC。
    	> +  设置 TIM3_CH2 的 PWM 模式，使能 TIM3 的 CH2 输出。
    	> +  使能 TIM3。
    	> +  修改 TIM3_CCR2 来控制占空比。

- 了解双路PWM的工作原理

  - 掌握技能

    > + 呼吸灯
    > + 直流电机控速	
    > + 双路PWM控制直流电机正转反转（可选)

## 7.第七项:

- 了解捕获的原理

- 了解特殊引脚的捕获及其通道设置

   - 掌握输入捕获设置的一般流程（以TIM5为例）

     	> + 开启 TIM5 时钟和 GPIOA 时钟， 配置 PA0 为下拉输入。
     	> +  初始化 TIM5,设置 TIM5 的 ARR 和 PSC。
     	> + 设置 TIM5 的输入比较参数，开启输入捕获
     	> + 使能捕获和更新中断（设置 TIM5 的 DIER 寄存器）
     	> + 设置中断分组，编写中断服务函数
     	> + 使能定时器（设置 TIM5 的 CR1 寄存器）

  - 掌握技能

    	> + 编码器测速
    	> + 超声波测距
    	> + 板载电容触摸捕获
